# Instructions for setting up

## Node version

Preferred: v16.14.0

## Steps:

Open terminal and enter

```bash
yarn install
```

- To run development version:

```bash
yarn start
```

- To run production version:

```bash
yarn start:prod
```

### Open http://localhost:3000/ on Chrome or your preferred browser engine
